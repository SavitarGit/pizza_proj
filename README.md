# Pizza Delivery Web Application

Не успел доделать функции измения количества пицц в корзине и удаления их из корзины.

Каждую пиццу можно добавить в корзину только один раз в любом порядке.

Корзина открывается нажатием на оранжевую кнопку / счетчик справа вверху. Закрывается только нажатием на крестик в самой корзине.
